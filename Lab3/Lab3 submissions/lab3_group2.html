Use the following: \[(1 +\mu \Delta_t + \sigma\epsilon \sqrt(\Delta_t)) = (1 + qnorm(epsilon, stock\_mu \times delta\_t, stock\_sigma \times sqrt(delta\_t))) \] Define delta_t as \(1/n\). Once done, test your function using f.stock.return(7000,20,0.05,0.2).
#Q3.b
f.stock.return = function(stock_price,n_periods, stock_mu, stock_sigma){
  delta_t = 1/n_periods
  epsilon = runif(n = 1, min = 0, max = 1)
  price = vector()
  price[1] = stock_price
  for(i in 2:(n_periods+1)){
    price[i] = price[i-1]*(1+qnorm(epsilon,stock_mu*delta_t,stock_sigma*sqrt(delta_t))) 
  }
  return(price)
}
